👩🏼‍🏫 CS Courses I Teach

Web Design & Development

HTML CSS JS

The goal of the Introduction to Computer Science course is to build a foundation in computational thinking and programming skills through the lens of Web Design & Development. We’ll review basic web design with HTML and CSS before developing interactive websites, applications, and games using JavaScript. In the process, students will expand their creativity, thoughtfulness, and problem-solving abilities.

Object-Oriented Programming (AP CSA)

JAVA

This course follows the College Board Advanced Placement curriculum for “AP Computer Science A.” The goal of the course is to design, analyze, and implement solutions to a wide array of problems. Students are introduced to the Java programming language as a means of accomplishing this goal. The technical focus of the course is Object-Oriented Programming (OOP) design and implementation. All students are expected to take the AP Computer Science A examination in May.

Advanced Topics in CS

PYTHON

The Post-AP Advanced Topics in Computer Science course offers highly motivated students the opportunity to expand on topics covered in the Introductory and AP courses. Modern applications of the Python language are highlighted, including Data Science & Exploratory Data Analysis (pandas, matplotlib, seaborn, etc.), creating Web Applications with Flask and Django, and Database operations with SQL.
